Detalhes do projeto:

1ª imagem: calcula a média dentre 3 números informados pelo usuário.

2ª imagem: sorteador de números, no qual o programa gera um número aleatório e o usuário vai digitando números até encontrar o número sorteado.

3ª imagem: utilização de uma API para busca e consulta de informações de filmes.

4ª imagem: dicionários da própria linguagem Python.

5ª e 6ª imagem: framework Pandas para inserção, modificação e exclusão de valores de um arquivo Excel.

7ª e 8ª imagem: utilização do framework Selenium, uma API e de um site para automatização de inserção de palavras em português aleatórias.

9ª e 10ª imagem: utilização de conectores entre MySQL e Python para inserção, modificação e exclução de valores do banco de dados.

11ª imagem: utilização de uma API para busca de significado e sinônimos de palavras informadas.

12ª imagem: envio de e-mail pelo python com utilização de suas bibliotecas.

13ª imagem: detecção da maior palavra informada retirando sinais de pontuação e deixando a palavra com a primeira letra maiúscula.

14ª imagem: utilizando classes do Python para determinar tipo, som e reprodução de animais.

15ª imagem: uso do Pytest para validar 2 casos da programação anterior.
